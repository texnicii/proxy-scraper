#### Get and save proxy list to CSV

Example
```
./cli
Found proxy: 2294
Stored in: proxy_2022-06-25-113230.csv
```

#### Usage as a package

- Import this package `go get -u github.com/texnicii/proxy-scraper`
- Call in your golang application
`proxyscraper.Get()`
